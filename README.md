Small math games launched in the console:
- brain even
- brain greatest common deliver
- brain prime
- brain calc
- brain progression

## Setup
`sudo npm link`

## Run game
`brain-even`
`brain-gcd`
`brain-prime`
`brain-calc`
`brain-progression`
